HTML

HTML是什么？

html是描述网页的一种语言。

HTML 指定的是超文本***标记***语言 (Hyper Text Markup Language)
HTML 不是一种编程语言,而是一种标记语言(Markup Language)
标记语言是一套标记标签
HTML使用标记标签来描述网页

书写规范

html结构

html标签

是以尖括号包裹的关键字成对出现的

有的开始标签,有结束标签,支持正确的嵌套

大部分标签有属性，格式： 属性一=”属性值1”

空标签，功能单一，不需要标签体，可以直接简写为<标签 />

html标签不区分大小写，但是建议小写
